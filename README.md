# Car Price Prediction - ML Service

Проект посвящён построению модели машинного обучения для предсказания стоимости автомобилей и разработке интерактивного сервиса на Streamlit.

## Что сделано

### Анализ данных
- Проведён первичный анализ данных: распределения признаков, статистики, корреляции.
- Обработаны пропуски, дубликаты и некорректные форматы числовых признаков.
- Выполнена визуализация ключевых зависимостей.

### Моделирование
- Обучены линейные модели: Linear Regression, Lasso, Ridge, ElasticNet.
- Проведена стандартизация признаков и OneHot-кодирование категориальных переменных.
- Подбор гиперпараметров выполнен с помощью GridSearchCV.
- Лучшая модель — Ridge-регрессия с категориальными и числовыми признаками.

### Оценка качества
- Использовались метрики R² и MSE для train и test.
- Реализована кастомная бизнес-метрика: доля прогнозов с ошибкой не более 10%.
- Показано сравнение моделей по бизнес-метрике.

### Streamlit-приложение
Реализован интерактивный сервис, который:
- отображает EDA-графики для загруженного CSV-файла;
- позволяет делать предсказания стоимости авто по CSV или ручному вводу признаков;
- визуализирует веса (коэффициенты) обученной модели.

### Оценка разработанного сервиса

Разработанное Streamlit-приложение является удобным и интуитивно понятным: пользователь может загружать CSV-файлы или вводить признаки вручную
для получения предсказаний стоимости автомобиля. Реализованы EDA-визуализации и наглядное отображение коэффициентов модели, что упрощает
интерпретацию результатов.
Основным ограничением сервиса является использование линейной модели, которая не всегда способна учитывать сложные зависимости в данных. В
дальнейшем качество сервиса можно улучшить за счёт применения более сложных моделей и расширения набора признаков.
